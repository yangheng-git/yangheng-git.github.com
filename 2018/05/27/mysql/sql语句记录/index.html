一张表的多次查询

项目中用到的语句， 首先查出指定用户所有的sessionId ， 然后用传入的sessionId 去取数据， 可能会遇到同一sessionId ，但是useriD 为0的情况， 所有进行判断。 然后将查出的数据进根据sessionId 分组。 取这个组中的创建时间 最大-最小。 得出 用户这次请求的持续时间（秒） ，取出必要的数据， 当时日期（天） 创建时间（时间戳）.。 第二次查询。 对取出的数据进行分组， 通过 “天” ， 然后取出每天中的多条在线时长进行sum. 得出每天的在线时长
